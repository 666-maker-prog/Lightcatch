# 变更日志

所有重要变更将记录在此文件中。

## [1.0.0] - 2026-07-03

### 项目初始化
- 从 JeecgBoot 提取 AI 模块，搭建独立 LightCatch 项目
- 定位：面向自媒体创作者的 AI 内容生产平台

### 已完成功能
- 用户系统：登录、注册、JWT 认证
- AI 对话：SSE 流式聊天，支持 DeepSeek/通义千问/OpenAI
- 素材库（RAG）：文档上传 → 文本提取 → 自动向量化 → 语义检索
- AI 写作：生成、改写、起标题、列大纲、优化、扩写、缩写
- 模型管理：支持对话/向量嵌入/图片/语音/视频 5 类模型
- 工作流：可视化拖拽编排 + LiteFlow 执行引擎
- 多租户：MyBatis-Plus 租户隔离

### 技术债
- PGVector 未安装，使用纯 Java 向量存储替代
- 工作流节点类型待扩展（图片生成、定时触发、条件分支）
